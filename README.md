# How to Create a Roblox Account
1. Press the link provided earlier, and in the top right, click **sign up**
2. Create a Username & Password, include your age and preferred gender (this will create your character, for this purpose, it does not matter)
3. Press play, and install Roblox, this should install both Roblox Studio & Roblox Player. For this purpose, you will use Roblox Player
4. When fully installed, press play!
